# Diabetes Risk Prediction Platform

A full-stack machine learning healthcare application that predicts diabetes risk using patient health metrics.

## Features

- Diabetes risk prediction using Machine Learning
- SHAP explainability for model transparency
- PDF report generation
- SQLite database storage
- Prediction history dashboard
- FastAPI backend
- React frontend

## Tech Stack

### Backend
- FastAPI
- Scikit-learn
- SHAP
- SQLite
- ReportLab

### Frontend
- React
- Vite
- Axios
- Tailwind CSS



## Project Architecture

Frontend (React + Vite)
        ↓
FastAPI Backend
        ↓
Machine Learning Model
        ↓
SHAP Explainability
        ↓
SQLite Database
        ↓
PDF Report Generator
